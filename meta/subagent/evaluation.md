# Subagent 评估策略

用于定义本仓库自检质量评估方法，支持横向比较与回归追踪。

## 评估目标

- 衡量验证结果的可信度与稳定性。
- 对比不同方案的执行成本与失败率。
- 为后续自动化提供结构化基线。

## 自检闭环

1. 在 `meta/docker/Dockerfile` 环境执行验证。
2. 用 `meta/subagent/acceptance.md` 做通过判定。
3. 将问题样例沉淀到 `meta/subagent/questions/`。
4. 用本文件指标复盘并更新下一轮策略。

## 建议指标

1. 成功率：验证步骤首次通过比例。
2. 复现率：同一流程重复执行结果一致比例。
3. 故障定位时长：从失败到定位根因的耗时。
4. 修复回归率：修复后再次失败的比例。
